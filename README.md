# CropGuard AI

**Smart Crop Disease Detection for Nigerian Farmers**

AI-powered crop disease identification with multilingual support (English, Yoruba, Igbo, Hausa, Pidgin).

## Features
- AI Pest and Disease Identification
- Weather-Based Crop Recommendations
- Community and Expert Support
- 5 Nigerian Languages
- PWA / Offline-First
- Voice Readout for accessibility

## Quick Start

### Frontend
Open frontend/index.html in your browser.

### Backend
cd backend
npm install
npm start

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express.js
- AI: Google Gemini Vision API
